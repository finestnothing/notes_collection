# self_describing_data_sources
Created: 20221101 17:50
Modified: 20221101 17:50

Self describing data sources are files that define column name and values in them at each stage
json is a good example
```json
{
	"name":"john",
	"age":30,
	"car":null
},
{
	"name":"bob",
	"age":85,
	"car":"Old AF"
}
```
